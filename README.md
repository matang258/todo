
# TodoApplication

할 일을 간단하게 기록하는 서비스  

### 특징
AWS Elastic Beanstalk을 이용하여 load balancer와 auto scaling group 구현
  

### 사용기술
▶️ FrontEnd
- React.js 18.2
- matreial-ui

▶️ Backend
- SpringBoot 2.7
- Gradle
- JPA
- H2
- MySQL

▶️ Deploy
- AWS elasticBeanstalk
- AWS EC2
